Logging¶

Current¶

the logging is done within the Event Handler. Every event triggers an entry in the log file. There are 37 columns but not all of them are relevant for each event. Only relevant data is logged as a number and other entries are saved as ‘nan’ to simplify reading of the log file.

  1. time

  2. global_pointer

  3. period_idx

  4. event_code

  5. pPre_0

  6. pDepre_0

  7. pPre_after_0

  8. pDiff_0

  9. tSwitchDepressure_0

  10. tSwitchDepressureEst_0

  11. tSwitchPressure_0

  12. tSwitchPressureEst_0

  13. gradientPressure_0

  14. gradientDepressure_0

  15. gradientPressureEst_0

  16. gradientDepressureEst_0

  17. riseTime_0

  18. fallTime_0

  19. pPre_1

  20. pDepre_1

  21. pPre_after_1

  22. pDiff_1

  23. tSwitchDepressure_1

  24. tSwitchPressure_1

  25. gradientPressure_1

  26. gradientDepressure_1

  27. fallTime_1

  28. riseTime_1

  29. period

  30. delay

  31. pressure_pulse_width

  32. depressure_pulse_width

  33. pump_stroke

  34. depressure_valve_counter

  35. pressure_valve_counter

  36. leak_value

  37. meanbit3
